### 软件工程个人项目



##### 命令行参数

| 参数                       | 参数意义                                         | 用法示例                                   |
| -------------------------- | ------------------------------------------------ | ------------------------------------------ |
| `-i <path to input file>`  | 带一个参数，表示输入文件的路径（绝对或相对路径） | `intersect.exe -i input.txt -o output.txt` |
| `-o <path to output file>` | 带一个参数，表示输出文件的路径（绝对或相对路径） | `intersect.exe -i input.txt -o output.txt` |
